SQL WHERE 子句

在Sql中Where的作用：满足指定条件的记录。如:>,=,<,<>或!=,>=,<=,BETWEEN,LIKE,IN,EXISTS,ALL,ANY 都在Where后面添加

BETWEEN

BETWEEN 描述：满足指定范围内 例子：Between 10 and 25 （在大于等于10小于等于25之间）

IN

IN 描述：操作数属于子查询中的值之一 例子：In (1,4,7) 或 in( select [字段] from[表]where[条件])

LIKE 模糊查询

LIKE 通配符：%,_(下划线),[],[^]

EXISTS

子查询有结果集（相关与非相关）
